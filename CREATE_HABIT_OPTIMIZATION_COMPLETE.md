## 📋 详细改动说明

### P0-1: 强化每天1次=最优

**文档原则:**
> 用户在这里最容易犯的错："我应该多做一点"

#### 改动1: 优化频次选项文案

**Before:**
```html
<view class="frequency-sub">(推荐)</view>
```

**After:**
```html
<view class="frequency-sub">（最容易坚持）</view>
```

**心理效果:** "推荐"是理性建议，"最容易坚持"是结果承诺

---

#### 改动2: 添加警告提示

**新增:**
```html
<view class="frequency-warning" wx:if="{{formData.target_times_per_day > 1}}">
  <text class="warning-icon">⚠️</text>
  <text class="warning-text">次数越多，越容易放弃</text>
</view>
```

**触发条件:** 选择2次/天或以上时显示

**样式特点:**
- 淡黄色渐变背景 (#fef3c7 → #fde68a)
- 黄色边框 (#fcd34d)
- 棕色文字 (#92400e)
- 警告图标

**对比效果:**
| 场景 | Before | After | 改进 |
|------|--------|-------|------|
| 选择1次 | (推荐) | （最容易坚持） | +肯定 |
| 选择2次 | 无提示 | ⚠️ 次数越多，越容易放弃 | +警示 |

**这是在替用户做正确决定**

---

**心理效果:**

- ✅ 鼓励用户迈出第一步

### P1-1: 预期完成率文案重写

**文档原则:**
> 从「预测成功概率」→「降低心理负担」

#### 改动1: 数据转文案

**Before:**
```html
<text class="impact-label">预期完成率:</text>
<text class="impact-value">85%</text>
```

**After:**
```html
<text class="impact-label-new">大多数人都能做到</text>
```

**转换规则 (create-habit.js):**
```javascript
let completionRateText = '';
if (expectedRate >= 80) {
  completionRateText = '大多数人都能做到';
} else if (expectedRate >= 60) {
  completionRateText = '有一定挑战，但可以试试';
} else {
  completionRateText = '建议从更简单的开始';
}
```

**样式特点:**
- 绿色文字 (#10b981)
- 字号稍大 (28rpx)
- 加粗显示

---

#### 对比分析

| 频次 | 原始数据 | 新文案 | 心理效果 |
|------|---------|--------|---------|
| 1次/天 | 预期完成率: 85% | 大多数人都能做到 | ✅ 社会认同 |
| 2次/天 | 预期完成率: 70% | 有一定挑战，但可以试试 | ✅ 鼓励尝试 |
| 3次/天 | 预期完成率: 55% | 建议从更简单的开始 | ✅ 善意劝阻 |
| 4次/天 | 预期完成率: 40% | 建议从更简单的开始 | ✅ 明确建议 |

**核心转变:**
- ❌ 数据评估（理性） → ✅ 情绪安抚（感性）
- ❌ "你可能失败" → ✅ "大家都成功"
- ❌ 被评判感 → ✅ 被支持感

### P1-2: 触发时间标题优化

**文档原则:**
> 把「决策」变成「直觉选择」

**对比效果:**

| Before | "选择触发时间" | 😕 需要思考 → 卡住 |

|  | "先选一个不容易忘的" | 💡 降低标准 → 减少纠结 |

**核心转变:**
- ❌ 系统指令 → ✅ 友好提问
- ❌ 需要决策 → ✅ 凭直觉选
- ❌ "正确答案" → ✅ "不容易忘"

## 💡 设计理念总结

### 核心转变

**从「逻辑正确」到「心理友好」**

| 维度 | Before（逻辑正确） | After（心理友好） |
|------|-------------------|------------------|
| **语气** | 系统指令 | 友好建议 |
| **数据** | 完成率85% | 大多数人都能做到 |
| **按钮** | 创建习惯（承诺） | 从今天试试（尝试） |
| **引导** | 选择触发时间 | 什么时候做最轻松？ |
| **警示** | 无 | 次数越多，越容易放弃 |

---

### 文档核心金句实践

> **"不是在创建一个长期计划，而是在给今天留一件小事。"**

**实践体现:**

1. ✅ 频次文案: "推荐" → "最容易坚持"（明确最优）
2. ✅ 频次警告: 添加黄色提示框（替用户做决定）
3. ✅ 创建按钮: "创建习惯" → "从今天试试"（降低承诺）
4. ✅ 底部提示: "随时可以改，不会清零"（消除焦虑）
5. ✅ 完成率: "85%" → "大多数人都能做到"（社会认同）
6. ✅ 触发时间: "选择" → "什么时候最轻松"（降低门槛）

1. **社会认同原理**
