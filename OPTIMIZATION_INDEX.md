## 🎓 关键概念解释

### "Tier X反馈"是什么？

按打卡时机和成就程度分级的反馈体系：

```
Tier 1 (日常打卡)    →  ✅ + 轻震 + "已完成"
Tier 2 (第3天)       →  ⭐ + 中震 + "坚持3天"
Tier 3 (第7天)       →  🔥 + 强震 + "首周达成" + 勋章
Tier 4 (恢复后)      →  💪 + 温震 + "欢迎回来"
```

### "关键节点干预"是什么？

根据用户在7天旅程中的不同阶段，主动推送恰当的激励：

```
Day 1  → 首卡激励（强）
Day 2-3 → 轻提示 + Day 3勋章
Day 5  → 分享激励
Day 7  → 庆祝 + 下周引导
```

### 为什么强调"30秒"？

```
30秒 = 完全降低心理门槛
      不需要思考，只需一个动作
      → 高频打卡 → 形成习惯 → 留存
```

## 💡 常见问题

### Q1：这些优化是否会拖累打卡速度？
**A**：不会。所有优化都是**非阻塞的**：
- 首卡激励：新用户首次，不影响日常
- 反馈弹层：异步显示，点击或自动消失
- 关键节点干预：可收起，不强制

### Q2：数据会不会变复杂？
**A**：反而简化了用户认知：
- 原版：只看完成率（很单调）
- 优化版：看成就感 + 对标数据（更激励）

### Q3：这套方案是否经过验证？
**A**：所有改进基于：
- ✅ 行为心理学（进度、成就、对比）
- ✅ 产品数据规律（用户流失高峰）
- ✅ 类似产品最佳实践（Duolingo、Strava等）

### Q4：如果用户讨厌过度反馈怎么办？
**A**：内置了3个人性化设计：
1. 反馈可点击跳过（不强制看）
2. 提示可收起（不打扰）
3. 文案温暖（不傲慢）
